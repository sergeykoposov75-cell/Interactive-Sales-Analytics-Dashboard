# Дашборд продаж — розничная компания

## Описание

Скрипт `generate_dashboard.py` генерирует синтетические данные о продажах розничной компании за 2025 год и создаёт Excel-файл `sales_dashboard.xlsx` с полноценным дашбордом для анализа.

## Требования

- Python 3.8+
- Библиотеки: `openpyxl`, `pandas`, `numpy`

Установка зависимостей:
```
pip install openpyxl pandas numpy
```

## Запуск

```
python generate_dashboard.py
```

На выходе появится файл `sales_dashboard.xlsx` в текущей папке.

## Структура Excel-файла

| Лист | Описание |
|------|----------|
| **Данные** | Сырые данные (~10 000 записей) с автрофильтром |
| **Сводка по регионам** | Агрегированные показатели по регионам (сортировка по убыванию выручки) |
| **Сводка по категориям** | Агрегированные показатели по категориям |
| **Динамика по месяцам** | Выручка и транзакции по месяцам |
| **Дашборд** | Интерактивная панель с KPI, диаграммами и фильтрами |

## Лист «Дашборд»

Содержит:
- **KPI**: общая выручка, количество транзакций, средний чек, количество регионов/категорий
- **Фильтры**: выпадающие списки для выбора категории и региона
- **Таблицы**: выручка по регионам (фильтруется по категории) и по категориям (фильтруется по региону)
- **Диаграммы**:
  - Линейная — динамика выручки по месяцам
  - Столбчатая — выручка по регионам
  - Круговая — доля категорий в выручке
- **Условное форматирование**: подсветка максимальных значений оранжевым

Диаграммы автоматически обновляются при изменении фильтров.

## Импорт в Google Sheets

1. Откройте [Google Sheets](https://sheets.google.com)
2. Создайте новую таблицу или откройте существующую
3. `Файл` → `Импорт` → `Загрузить` → выберите `sales_dashboard.xlsx`
4. Выберите «Заменить текущую таблицу» или «Создать новую таблицу»
5. Нажмите «Импортировать»

### После импорта

- **Данные**, сводки и формулы SUMIF/SUMIFS работают без изменений
- **Диаграммы** Excel не импортируются в Google Sheets автоматически.
  Для воссоздания диаграмм:
  1. Перейдите на лист «Дашборд»
  2. Выделите данные для диаграммы (например, столбцы Месяц и Выручка)
  3. `Вставка` → `Диаграмма`
  4. Выберите тип (линейчатая, столбчатая, круговая)
  5. Настройте диапазон данных и подписи
- **Условное форматирование** и **фильтры** работают в Google Sheets

### Примечание по фильтрам

После импорта в Google Sheets фильтры на листе «Дашборд» (ячейки B6 и E6)
продолжают работать — таблицы с формулами SUMIF/SUMIFS пересчитываются
автоматически при изменении значения фильтра.

## Параметры генерации

- Сид (seed): **42** (фиксирован для воспроизводимости)
- Выручка: от 1 000 до 50 000 руб.
- Транзакции: от 1 до 50
- Записей в день: от 5 до 45 (с учётом сезонности и выходных)
- Регионов: 5, Категорий: 5
