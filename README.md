# Interactive-Sales-Analytics-Dashboard
Интерактивный дашборд для отдела продаж крупной розничной компании. Обработаны предоставленные данные о продажах за последний год, используя Python и инструменты аналитики данных, чтобы выявить ключевые тенденции и аномалии. Дашборд реализован в Google Sheets с использованием возможностей визуализации для удобного представления информации.
